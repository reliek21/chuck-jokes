
# Chuck Jokes

Chuck Jokes, is a small project that uses the api from [chucknorris.io](https://api.chucknorris.io/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/reliek21/chuck-jokes.git
```

Go to the project directory

```bash
  cd chuck-jokes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
